# 躲猫猫🐱——Go
### 启航：

Go语言开发的以Gin为基础的躲猫猫全平台后端服务。目前后端为Koa2，正逐步迁移至Go.

- 躲猫猫官网：React

- 躲猫猫APP：Flutter

- 躲猫猫下载器: C++、C

本项目结合CI/CD、Docker作为持续迭代。综合Redis、Mongodb、mysql、ES、RabbitMQ、阿里云OSS、分布式，最后升级至微服务。

躲猫猫平台是个年轻人短视频娱乐平台，所以，会加入Ffmpeg、WebRTC、OpenCV，并加入推荐算法和Tensorflow。

躲猫猫全平台均由我一个人工作之余进行开发，故开发周期长。

持续进行中，预计10月1日前上线全平台产品，有疑问请提交对应仓库的issue! 

任务进度请查看仓库的Project看板
